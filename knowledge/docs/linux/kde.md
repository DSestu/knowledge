# KDE

## Useful shortcuts

* ctrl + win + arrow: change virtual desktop
* ctrl + win + shift + arrow: move to other virtual desktop with the current window
* "Toogle grid view": Set it to win + tab
* "Toogle overview": Set it to win + w
* "Keep window on all desktops": Set it to win + x

## Define session-wide shortcut to switch between windows

Either manually:

* Alt+F3 to bring the window menu > More actions > Set window shortcut

Either customize the create shortcut shortcut:

* Shortcuts > Window Management > Setup Window Sortcut

I personally setup shortcut with `meta + ²`, then use `meta + 1/2/3` as custom shortcuts.
